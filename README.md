# loveTank
几何坦克大战

基本玩法：
- WASD控制坦克上下左右移动
- QE控制炮弹方向（顺逆时针旋转）
- J发射炮弹，每次10发子弹
- K填充炮弹，填充时需要停顿三秒

游戏目标：
保护己方基地，摧毁敌方基地
支持双人
